ℹ️ OBS Setup

Browser Source Recommended

  1. Enter or generate a Room Code in the tracker, then click Connect
  2. Copy the OBS URL that appears
  3. In OBS add a Browser Source
  4. Paste the OBS URL. Default size is 315 × 450
  5. Add &scale=1.5 to the URL and multiply the source size by that scale if you want it larger
  6. The overlay background is transparent — no chroma key needed

Popup Window

Click Open OBS Overlay to open a 315×450 popup then capture it with Window Capture in OBS.
